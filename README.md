# Dependencies
- Node 21
- Vue 3.4
- Vite 5.1
- Typescript 5.2
- Sass 1.71

## How to use it

It's fairly simple, just clone the repo with `git clone`, install the deps with `yarn` and you should be good to go. You just need to `yarn dev` after that !
